Class Variables

and Instance Variables

April 5, 2015

Class methods do not have access to instance methods or instance variables. You can use class variables to store data that belongs to a class, but not to its instances. Class instance variables are available only in class methods, while class variables are available in both class methods and instance methods.

A class variable is prefixed with @@ symbols before its name. Instance methods can access both class methods and class variables. An instance variable is prefixed with just one @ symbol. Class instance variables use the same notation as that of an instance variable. But unlike instance variables, you declare them inside the class definition directly.

You will rarely need to use class variables, and you should very rarely use them to store state. There are only a few cases in which class variables are the right choice, like when you need store data at a class level. You may use them to store application configurations like the name, version, database and other settings. Use instance variables instead of class variables when you don't want your data to be shared across the inheritance chain.
